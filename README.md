Lumen-Interactive Light-Based Web Experience
About:
Lumen is an interactive web-based project that creates a dynamic light structure experience using animations and visual effects. It focuses on creativity, motion, and modern frontend design principles.

Features:
Interactive light animation system
Smooth visual effects
Responsive design
Creative UI/UX concept
Built With
HTML
CSS
JavaScript

Purpose:
This project was created to explore creative frontend development, animations, and interactive visual design using core web technologies.
